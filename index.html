Meus Projetos

Elementos Textuais

O objetivo deste projeto foi criar um blog simples, utilizando HTML e CSS, com ênfase na formatação e no uso adequado de elementos textuais.

Carrossel

O objetivo deste projeto foi criar um carrossel de imagens interativo, utilizando HTML e CSS, para exibir uma coleção de fotos ou imagens em um formato deslizante, permitindo ao usuário navegar pelas imagens de maneira fluida e intuitiva.

Uso das Redes

O objetivo deste projeto foi criar um site interativo para analisar e exibir dados sobre o uso excessivo das redes sociais, utilizando tabelas para mostrar informações estatísticas, gráficos e resultados de pesquisas.

Formulário

O objetivo deste projeto foi criar um formulário de contato interativo e funcional para um site, utilizando HTML e CSS. O formulário deveria permitir que os usuários enviassem informações como nome, e-mail, mensagem, e opções de contato.
